GIDGraph

Gene Interaction Description
ⓘ

Input natural language discriptions of gene interactions into this field. From here, the interactions can be extracted, and displayed in the Semi-Natural Language box.

Semi-Natural Language
ⓘ

In this field you can add, remove and edit gene interactions. You can do this by clicking on the different elements. These changes will be reflected in the visualisation, but changing elements here will not impact the natural language discriptions.

Regulatory Network
ⓘ

Here is the visual representation of the gene interactions presented in the Semi-Natural Language field. The graph is interactive.

Logical Formulas
ⓘ

In this field, the logical formulas derived from the gene interactions is shown. Click the 'and' and 'or' buttons to switch between. The user can export these logical formulas to GINML, for use in GinSim.
